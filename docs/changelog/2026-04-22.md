## Batch 3 (feat/batch-3-polish · Later · 완성도 향상)

### Foundation (T-14·T-13·T-17 — 인프라)

- **T-17** · `login.html` 의 `scrollIntoView` 를 `getBoundingClientRect + scrollTo` 로 치환. iOS Safari + fixed header + safe-area 조합에서 레이아웃 jank 방지.
- **T-14** · 폰트 패밀리 4→2. DM Sans · Noto Serif KR 제거, Cormorant Garamond + Noto Sans KR 유지.
  - `tokens.css` / `tokens-tv.css` 의 `--font-heading / --font-body` 재정의
  - 11개 HTML 의 Google Fonts `<link>` 에서 DM Sans 제거
  - 하드코딩된 `font-family: 'DM Sans'|'Cormorant Garamond'` 100+ 건을 `var(--font-body)` / `var(--font-heading)` 로 sweep
  - 페어링 코드 등 monospace 용도는 `ui-monospace, 'SF Mono', Menlo, Consolas, monospace` 시스템 스택으로 교체
  - `Noto Sans KR` 선택 이유: 신규 CDN 요청 없음 (기존 로드 활용). Pretendard 로 교체 여부는 후속 판단.
- **T-13** · Twemoji SVG 브랜드 핵심 위치에만 적용. `twemoji-brand.js` 신규 (jsdelivr CDN + MutationObserver 로 동적 재파싱).
  - 적용 대상: `login.html` .hero · `create.html` #packGrid · `presenter.html` header/main · `result.html` #share-card/#storyCanvas
  - 본문·채팅·게스트 입력 이모지는 OS 기본 유지 (페이로드 절감)
  - CDN 실패 시 graceful degradation (OS 이모지 폴백)
  - `result.html` 의 share-emoji 는 Twemoji 가 textContent 를 `<img>` 로 치환 → dataset 에 원본 이모지 보관해 `buildStoryCanvas()` 에서 복원
